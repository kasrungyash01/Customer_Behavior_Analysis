# Customer Behavior Analysis

## Overview

This project focuses on analyzing customer behavior data to uncover purchasing patterns, customer preferences, and key business insights. The analysis was performed using Python, PostgreSQL, and Power BI, followed by the creation of a business report and presentation.

The objective of this project is to transform raw customer data into actionable insights that can support data-driven business decisions.

---

## Dataset

The dataset contains customer-related information, including:

* Customer demographics
* Purchase history
* Product categories
* Transaction details
* Customer engagement metrics

**Dataset Format:** CSV

## Project Workflow

### 1. Data Loading

* Imported CSV dataset into Python.
* Performed initial inspection of data structure and quality.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Standardized column formats.
* Corrected inconsistent data entries.

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Identified purchasing trends.
* Examined product preferences.
* Generated visualizations to uncover patterns and insights.

### 4. PostgreSQL Analysis

* Created database tables.
* Imported cleaned dataset into PostgreSQL.
* Executed SQL queries to analyze:

  * Customer segments
  * Sales performance
  * Purchase frequency
  * Revenue trends

### 5. Power BI Dashboard

* Connected PostgreSQL data to Power BI.
* Built interactive dashboards and KPI visualizations.
* Created filters and drill-down reports for detailed analysis.

### 6. Reporting & Presentation

* Prepared a business insights report.
* Designed a professional presentation using Gamma.
* Summarized findings and recommendations for stakeholders.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL
* Power BI Desktop

### Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/customer-behavior-analysis.git
```

2. Install required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

3. Load the dataset into PostgreSQL.

4. Execute SQL scripts from the `SQL` folder.

5. Run the Python analysis notebook.

6. Open the Power BI dashboard file (`.pbix`) to explore visualizations.
